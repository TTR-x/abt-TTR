# **App Name**: TTR Gestion Ambassadeur

## Core Features:

- Ambassador Authentication: Secure login via email or ambassador code using Firebase Authentication.
- Personalized Referral Link Generation: Automatically generate a unique referral link for each ambassador (https://ttrgestion.com/?ref=CODE).
- Referral Tracking and Commission Calculation: Track referred clients and calculate commission (15% per active client) using data from the TTRGESTION API.
- Ambassador Level and Point System: Calculate ambassador level, and associated progress toward next level, according to points earned.
- API Integration: Fetch data from the TTRGESTION API (GET /clients?ref=CODE) to update the ambassador dashboard with client information.

## Style Guidelines:

- Primary color: Deep indigo (#1E3A8A) to convey trust and professionalism. Inspired by the importance of a dependable dashboard.
- Background color: Light grey (#F9FAFB) for a clean and readable interface.
- Accent color: Vibrant orange (#F97316) for call-to-action buttons to draw attention and create a sense of energy.
- Headline font: 'Poppins', a geometric sans-serif to create a precise, contemporary, fashionable look. This font is also suitable for short chunks of text.
- Dashboard layout providing key metrics at a glance, and the layout will adjust according to mobile or desktop
- Simple, intuitive icons that are easy to understand for users in diverse African markets.
